    Stock Price Movement Prediction Using LSTM and CNN Models

    • What I like about this project:
      1. A cool user interface.
      2. The user can search and select any company that they like.
      3. The models will be trained on the data of the selected company, so the prediction will be more precise as the dataset is not general.
      4. The user can modify some features to customize the models.
